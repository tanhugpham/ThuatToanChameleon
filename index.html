Giới Thiệu

Thuật toán Chameleon là một thuật toán phân cụm phân cấp dựa trên đồ thị, phù hợp với việc phân tích dữ liệu khách hàng.

Các Bước Thực Hiện

  1. Xây dựng đồ thị k-láng giềng gần nhất
  2. Phân chia ban đầu thành các cụm nhỏ
  3. Gộp cụm dựa trên độ tương đồng
  4. Đánh giá kết quả phân cụm

Công Thức Quan Trọng

Độ tương đồng giữa hai cụm:

RC(Ci,Cj) = RI(Ci,Cj) × RC(Ci,Cj)²

Trong đó:

  • RI: Độ kết nối tương đối
  • RC: Độ gần tương đối

Phân Khúc Khách Hàng

Phân tích hành vi mua sắm để chia khách hàng thành các nhóm:

  • Khách hàng thân thiết
  • Khách hàng tiềm năng
  • Khách hàng không thường xuyên

Ứng Dụng Thực Tế

  • Chiến lược marketing cá nhân hóa
  • Tối ưu hóa chuỗi cung ứng
  • Dự đoán xu hướng tiêu dùng
  • Phát triển sản phẩm mới

Hướng Dẫn Sử Dụng

  1. Chuẩn bị file CSV chứa dữ liệu khách hàng
  2. Tải lên file dữ liệu
  3. Điều chỉnh tham số k và kích thước cụm
  4. Nhấn "Bắt Đầu Phân Cụm"
  5. Phân tích kết quả trực quan
